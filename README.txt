The folder contains the table of differentially expressed proteins described in the  paper Nassa et al., 
and the R scripts used to create the VolcanoPlot showed in the  Figure 2a.
"Proteins_diff_exp.txt" contains the list of differentially expressed proteins
with the associated q-value and difference computed using PERSEUS (https://www.biochem.mpg.de/5111810/perseus).
Perseus analysis was done with freely available software, version 1.6.1.3 and the Perseus .sps file having details of
the analysis workflow is available upon request (email to: aweisz@unisa.it or rtarallo@unisa.it).
